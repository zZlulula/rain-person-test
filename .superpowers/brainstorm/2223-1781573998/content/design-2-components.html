🔘 GazeChoiceButton · 视线选项按钮（重点改造）

当前是纯静态样式切换。改造为三层动效叠加：

💧 雨滴反馈（每击必有）

用户点击/视线选中时触发

1. scale: 1.0 → 0.94（60ms）→ 1.03（200ms easeOutBack）→ 1.0（150ms）
2. 涟漪：以触点为中心，径向渐变扩散 400ms
3. InkWell.splashColor → 改为自定义 _ripplePainter

实现：GestureDetector + AnimatedScale + TweenAnimationBuilder

💡 霁光绽放（选中瞬间）

从未选中 → 选中时触发一次

18 颗光粒子从按钮边缘爆发：
· 颜色：accent (#7a9b8a) + 白
· 大小：2-5px，随机
· 方向：360° 均匀分布 ± 随机偏角 15°
· 距离：30-70px，随机
· 时长：1.2s，easeOutExpo
· 透明度：1.0 → 0，scale 1.0 → 0

实现：CustomPainter 叠加层 + AnimationController 单次播放

🌫️ 雾气显形（出现时）

按钮首次出现在屏幕上时

opacity: 0 → 1
translateY: 12 → 0
blur: ImageFiltered 6px → 0
时长：500ms，easeOutExpo
延迟：按索引 stagger 80ms

实现：TweenAnimationBuilder + ImageFiltered + 延迟 Timer
