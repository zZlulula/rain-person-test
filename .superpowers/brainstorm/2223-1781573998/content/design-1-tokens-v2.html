设计 v2 · 第一节：动画令牌 & 氛围系统（增强版）

从"能用"升级到"难忘"——雨境主题贯穿所有动效

🌧️ 核心隐喻：雨的五种形态 → 五种动效

不再用抽象的 duration/curve 令牌，而是用雨的具象形态命名，每种形态对应一组动画参数，贯穿全应用。

💧 雨滴 · 轻触反馈

按钮按下 → 像雨滴落入水面

scale: 1.0→0.96→1.02→1.0
duration: 400ms · easeOutBack
+ 涟漪扩散 (InkWell 自定义)

🌫️ 雾气 · 浮现过渡

文字/卡片出现 → 像晨雾中显形

opacity 0→1 + translateY 12→0
duration: 500ms · easeOutExpo
+ 模糊滤镜渐消 (ImageFiltered)

💨 微风 · 扫过切换

页面过渡 → 像风吹开雨帘

slideX + fade 组合
duration: 500ms · easeInOutCubic
方向跟随页面栈前进/后退

☁️ 云层 · 呼吸循环

等待/加载状态 → 像云层缓缓流动

opacity 0.4→0.8→0.4 循环
duration: 2500ms · easeInOutSine
替代默认 CircularProgressIndicator

💡 霁光 · 高亮绽放

选项选中 → 像雨后初晴的光

bgColor 渐变 + 柔光扩散
duration: 600ms · easeOutExpo
boxShadow 从 0 到扩散 halo

📐 技术实现

app_theme.dart 新增 RainMotion 类
/// 雨的 5 种动画预设
class RainMotion {
  // 雨滴 — 按钮按下反馈
  static Widget raindrop(child, onTap)
    → _RaindropWrapper
  // 雾气 — 浮现效果
  static Widget mistReveal(child, {delay})
    → _MistRevealWrapper
  // 微风 — 扫过过渡
  static Widget breezeSlide(child, dir)
    → _BreezeTransition
  // 云层 — 呼吸循环
  static Widget cloudBreathe(child)
    → _CloudBreatheWrapper
  // 霁光 — 高亮绽放
  static Decoration sunlight(active)
    → BoxDecoration + halo shadow
}

新方案不再仅仅是"参数常量"，而是用雨的五种形态作为隐喻统一全应用动效语言——按钮反馈如雨滴、文字浮现如晨雾、页面切换如风过、加载等待如云流、选中高亮如霁光。
