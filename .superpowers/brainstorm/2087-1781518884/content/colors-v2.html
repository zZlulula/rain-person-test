配色方案 v2 — 雨中意境 · 克制治愈

围绕"雨天、心理、治愈"三个关键词，克制不张扬

A

雨雾灰调（推荐）

深灰蓝底 + 柔和月光白 + 朦胧蓝灰卡片。像窗外细雨，安静不打扰。最契合"雨中"主题。

背景深灰蓝 → 卡片微亮 → 文字暖白 → 点缀蓝灰

B

暖灰石纹

暖灰棕底 + 奶油白文字 + 淡琥珀点缀。像雨天壁炉旁的温暖安全感，适合心理咨询氛围。

背景暖棕 → 卡片微亮 → 文字奶油 → 点缀琥珀

C

月夜靛蓝

靛蓝黑底 + 淡蓝白文字 + 银灰点缀。雨夜月光穿过云层的清冷高级感，克制优雅。

背景靛蓝 → 卡片微亮 → 文字蓝白 → 点缀银灰

D

云层灰白

浅灰白底 + 深灰文字 + 淡蓝绿点缀。像阴天雨前的云层，轻盈不压抑，偏亮色系。

背景云灰 → 卡片浅灰 → 文字深色 → 点缀青蓝
