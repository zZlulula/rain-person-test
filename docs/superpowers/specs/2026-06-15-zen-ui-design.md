# 禅意灰绿 UI 设计规格

**日期**: 2026-06-15  
**版本**: 未配色版 → 禅意灰绿  
**原则**: 整体逻辑和交互方式不变，仅优化视觉

---

## 1. 配色方案

| 用途 | 色值 | 说明 |
|------|------|------|
| 页面背景 | `#e8ebe4` | 浅灰绿，替代纯黑 |
| 卡片 / 按钮默认 | `rgba(58,74,63,0.05)` + 边框 `rgba(58,74,63,0.1)` | 微妙的层次 |
| 主文字 | `#3a4a3f` | 深灰绿，替代纯白 |
| 次要文字 | `rgba(58,74,63,0.5)` | 阶段标签、提示 |
| 蒙版1（引导） | `rgba(0,0,0,0.45)` | 不变，叠加在视频上 |
| 蒙版2（最终过渡） | `Colors.black`（100%） | 不变，"正在生成报告" |
| 按钮高亮 | `#7a9b8a` | 淡青绿，后端选中反馈 |
| Scafold 背景 | `#e8ebe4` | 全局 |

## 2. 排版调整

| 元素 | 当前 | 改为 |
|------|------|------|
| 标题字重 | `FontWeight.bold` | `FontWeight.w300`（纤细感） |
| 标题间距 | 无 | `letterSpacing: 2-6` |
| 按钮字重 | `FontWeight.w600` | `FontWeight.w300` |
| 报告卡片背景 | `Color(0xFF1A1A1A)` | `rgba(58,74,63,0.05)` |
| 报告卡片边框 | 无 | `1px solid rgba(58,74,63,0.08)` |

## 3. 组件修改

### 3.1 按钮（开始、确认、重新体验）
- 线条风格：透明底 + 细边框 + 轻圆角(12px)
- 正常态：`border: 1px solid rgba(58,74,63,0.25)`，`background: rgba(58,74,63,0.03)`
- 高亮态：`background: #7a9b8a`，`color: white`

### 3.2 GazeChoiceButton（视线选项按钮）
- 正常态：透明底 + 细边框
- 高亮态：填充 #7a9b8a
- 字体纤细 light/300

### 3.3 报告卡片
- 毛玻璃效果：半透明背景 + 浅边框
- 间距加大 padding: 16→20px

### 3.4 蒙版
- 蒙版1（引导阶段）不变：`Color.fromARGB(115, 0, 0, 0)` ≈ 45%
- 蒙版2（最终过渡）不变：`Colors.black`
- 两者的文字保持白色
