# 清透水晶 UI 交互特效设计

**日期**: 2026-06-16
**基于**: 禅意灰绿主题 (2026-06-15)
**原则**: 仅修改 UI 样式和交互动效，所有视频播放、定时器、后端调用逻辑不变

---

## 1. 动效语言：雨的五种形态

| 名称 | 用途 | 参数 |
|------|------|------|
| 💧 雨滴 | 按钮按下反馈 | scale 1→0.96→1.04→1.0, 400ms easeOutBack |
| 🌫️ 雾气 | 文字/按钮浮现 | opacity+fadeUp(16px)+blur(6→0), 600ms easeOutExpo, stagger 80-120ms |
| 💨 微风 | 页面过渡 | slideX(30px)+fade, 500ms easeInOutCubic, 方向跟随前进/后退 |
| ☁️ 云层 | 加载等待 | opacity 0.1↔0.8 呼吸循环, 2800ms easeInOutSine |
| 💡 霁光 | 选中高亮 | 粒子星芒爆发 + 双层涟漪 + 流光持续脉动 |

## 2. 水晶按钮 (crystal button)

### 正常态
- 0.5px 边框 `rgba(58,74,63,0.1)` + 0.5px 内描边 `rgba(255,255,255,0.3)`
- 对角线渐变背景：白→透明→白，模拟冰面折射
- 内阴影 inset 模拟顶部受光

### Hover
- scale 1.02
- ::before 流光从左到右扫过 (800ms)
- ::after 底部光晕亮起

### Press
- scale 0.96 (150ms)

### 选中态
- scale 1.04
- 绿色调渐变背景 `rgba(122,155,138,0.12-0.2)`
- 三层外发光环 (3px/10px/22px)
- 流光持续呼吸脉动 (3s 循环)
- 24 颗光粒子向四周爆发 (1.5s easeOutExpo)
- 双层涟漪扩散 (0.9s + 1.2s)

### 暗色背景变体
- 边框 `rgba(255,255,255,0.1)`
- 背景 `rgba(255,255,255,0.01-0.07)`
- 其余同浅色版
